Réalisation
Zelda.

Description du Projet

Création d'une application (Wiki) en C# sur le thème de Zelda Skyward Sword.

L'application permet de consulter une base de données SQLite contenant des informations sur les personnages, objets, lieux et quêtes du jeu. Elle a était réalisée en architecture MVVM (Model-View-ViewModel) sous Visual Studio 2026 en équipe de deux.

C# .NET Visual Studio 2026 SQLite Git/GitHub MVVM Android Studio

Travail Réalisé

1. Centralisation du code (GitHub)

2. Architecture MVVM

Organisation du code pour séparer la Vue de la Logique.

3. Base de données (SQLite)

4. Rendu sur Android

L'application finale permet de naviguer entre l'accueil, la liste et les détails.
